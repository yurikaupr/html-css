Veremos abaixo alguns testes com os textos em negrito e em itálico

Texto em negrito

Para deixar o texto marcado, basta usar a tag B. (forma nao semântica).

Ou, podemos utilizar a tag STRONG para marcar o nosso texto. (forma semântica).

Texto em itálico

Para deixar o texto no formato itálico, basta utilizar a tag I. (forma não semântica).

Ou, podemos utilizar a tag EM para marcar o nosso texto. (forma semântica).

Texto marcado

Para deixar o texto marcado, basta utilizar a tag MARK.

Para mudar a cor individualmente, basta utilizar a tag STYLE dentro do seu MARK.

Caso queira mudar a cor de todos os MARK, basta usar a tag STYLE dentro do seu titulo.

Texto deletado

Para "deletar" um texto, basta usar a tag DEL.

Texto sublinhado ou inserido

Para sublinhar um texto ou inserir, basta usar a tag INS.

Texto sobrescrito e subescrito

Para colocar o texto sobrescrito, basta utilizar a tag SUP.

Para colocar um texto subescrito, basta utilizar a tag SUB.
